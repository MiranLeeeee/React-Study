## Week1

### React 프로젝트 시작
* nvm 설치
    - LST버전 설치: `nvm install [LST버전]`
    - nvm version 사용:`nvm use [버전]`
* NPM(Node Package Manager): 다른 개발자 등이 개발한 패키를 활용할 수 있음
    - yarn과 비슷하며 node.js를 설치하면 따로 설치하지 않아도 됨
    - npm으로 yarn 설치: `npm install -g yarn`
    - yarn으로 패키지 설치: `yarn add [패키지]`
    - CRA(Create React App): 웹 만들 때 필요한 것을 모두 넣은 패키지
* 프로젝트 생성
    - 프로젝트 생성: `yarn create react-app [프로젝트]`
    - node_modules: 수많은 패키지를 가지고 있는 저장소
    - 시작하기: `yarn start`
